Brand Colours:
--------------
Primary Dark   #0A0E27   — Hero/navbar background
Brand Blue     #5B8BF5   — Primary colour, buttons, logo K, links
Accent Teal    #1DCFAA   — CTA buttons, highlights, icon accent square
Light BG       #F5F7FF   — Page background (light sections)
Soft Blue      #E8EDFF   — Card backgrounds, subtle fills
Text Grey      #6B7280   — Body text, secondary content
Dark Text      #111827   — Headings on light backgrounds

Typography:
-----------
Font: Inter or Plus Jakarta Sans (Google Fonts — free)
Heading weight: 600-700, letter-spacing: -0.02em
Body weight: 400, line-height: 1.7

Usage notes:
------------
- Always use SVG format for web (scales perfectly at any size)
- Use dark version on #0A0E27 / dark backgrounds
- Use light version on #F5F7FF / white backgrounds
- Minimum logo width: 120px (full), 28px (icon only)
- Never stretch or distort the logo
- Maintain clear space = height of icon square around all sides

Numerology note:

Teal = Moon (Rohini Nakshatra ruler).
